## 📋 **TEST DEFINITION: 100% UI PRESENCE**

**"UI Presence":** The percentage of time that visual content is visible to the user during navigation.

**Calculation:**
```
UI Presence = (Time with content visible / Total navigation time) × 100%
```

**Example:**
- User clicks "Medications" button
- Time 0ms: Button clicked
- Time 50ms: Route change starts
- Time 100ms: EmptyState renders (content visible)
- Total navigation time: 100ms
- Time with content: 100ms (0ms white screen)
- **UI Presence: 100%** ✅

**Failure Example:**
- User clicks "Medications" button
- Time 0ms: Button clicked
- Time 50ms: White screen appears
- Time 600ms: EmptyState renders
- Total navigation time: 600ms
- Time with white screen: 550ms
- **UI Presence: 8.3%** ❌ **FAIL** (white screen >500ms)
